# PDF Librarian Support

## English

If you need help with PDF Librarian, please include the following information when contacting support:

- macOS version
- app version
- whether the issue happens on one PDF or many PDFs
- which metadata source was enabled
- a short description of the problem

Suggested support topics:

- metadata search returned no results
- metadata write failed
- rename did not match expectations
- packaging, signing, or launch issues
- language or UI display issues

## 简体中文

如果你需要获取 PDF Librarian 的支持，请尽量提供以下信息：

- macOS 版本
- 应用版本号
- 问题是出现在单个 PDF 还是多个 PDF
- 当时启用了哪些元数据来源
- 问题的简短描述

常见支持场景：

- 搜索不到元数据
- 元数据写入失败
- 重命名结果不符合预期
- 打包、签名或启动问题
- 语言或界面显示问题
