# App Store 截图拍摄清单

建议先做两套语言：

- English
- 简体中文

## 建议截图顺序

1. 首页总览
   - 展示四步流程、当前主题、主标题

2. 联网检索候选结果
   - 展示多来源候选列表和已选候选

3. 第三步字段编辑
   - 展示 Dublin Core 字段可编辑、可勾选写入

4. 第四步重命名确认
   - 展示基于最新元数据生成文件名，并可手动修改

5. 多语言与外观切换
   - 展示国际化和浅色/深色界面

## 拍摄建议

- 不要放空白状态
- 用真实但不敏感的样例 PDF
- 每张图只讲一个能力点
- 第一张图要最容易让人秒懂产品用途
- 中英文两套截图内容结构保持一致
